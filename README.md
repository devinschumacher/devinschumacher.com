# devinschumacher.com

Next.js site for devinschumacher.com.

## Content

Content is stored directly in the repository:

- Blog posts: `content/blog/**/*.md`
- Product/content helpers: `lib/*.ts`
- Static routes: `app/**/page.tsx`

## Local Development

```bash
npm run dev
```

## Build

```bash
npm run build
npm run build:cloudflare
```
